รวมคำสั่งgit

git init = สร้าง local git repository
การเพิ่มไฟล์เข้าสู่ staging area (check-in) แบบต่างๆ
git add = เพิ่มไฟล์เดียวโดยการระบุชื่อไฟล์ เช่น git add index.html
git add *.html = เพิ่มหลายๆไฟล์โดยการระบุนามสกุลไฟล์
git add . = เพิ่มทุกไฟล์ที่มีใน Directory
การลบไฟล์ออกจาก Git Repository
git rm -r —cached . = ลบทั้งหมด
git rm —cached = ลบแบบระบุชื่อไฟล์
git status = ตรวจสอบสถานะ
git commit -m "Log Message" = เก็บประวัติการแก้ไข
git log = ดูประวัติการ commit
git log --online = แสดงแต่ละlog ในบรรทัดเดียว
git log --graph = แสดงเป็นเส้นbranch ให้ดูง่ายขึ้น
เปรียบเทียบversion (diff)
git diff = ใช้สำหรับความแตกต่างของแต่ละ commit
git checkout = ย้อนกลับไป versionล่าสุด
ย้อนกลับก่อน add หรือย้อนกลับไปcommit idที่ระบุ
git reset
git reset --option
soft ใช้ลบ commit ทั้งหมดที่อยู่หลังcommit id แล้วนำไฟล์ที่เคยอยู่ในcommit นั้น กลับมา staging area
mixed soft ใช้ลบ commit ทั้งหมดที่อยู่หลังcommit id แล้วนำไฟล์ที่เคยอยู่ในcommit นั้น กลับมา working directory
hard ใช้ลบ commit ทั้งหมดที่อยู่หลังcommit id แล้วทำลายไฟล์ที่เคยอยู่ในcommit นั้น
git branching
git branch = การแสดงรายชื่อ branch
git checkout -b <ชื่อ branch> = การสร้าง branch (ห้ามตั้งชื่อเว้นวรรคให้ใช้ - หรือ _ แทน)
git branch -d <ชื่อ branch> = การลบ branch
git checkout <ชื่อ branch> = การสลับ branch
git checkout master = กลับไป branch หลัก
git merge <ชื่อ branch> = การรวม branch
git push = อัพไฟล์จาก local repository ขึ้นไปที่ github
git pull = ดึงไฟล์จาก github ลงมา local repository
